毛周角化症俗称鸡皮肤

病症成因

  • 遗传 原因暂不明， 可能 与 （丝氨酸蛋白相关酶的基因缺陷），缺乏亚油酸、VC、VA等有关。由于原因不确定，故还难以说根治。 治不好，能改善。 可以改善到基本看不出来，但是需要长期的坚持。
  • 后天的则是由于皮肤过度干燥、受刺激过度，屏障受损导致的暂时性鸡皮肤。 比如暴力脱毛，再比如冬天天气特别干，还不涂润肤。

治理方法

主要方法是用水杨酸、果酸、蛋白酶等去角质，用维A酸类减缓角质化（所以你也可以吃富含胡萝卜素的食物）；用尿素软膏保持角质层水分，并注意防晒。激光脱毛可能有效，但目前还缺乏临床研究。
调理
1、加强锻炼大量排汗 运动完会立刻感觉到皮肤因血液循环而变白变红润；
2、日常护理/dd>

加强锻炼、坚持身体乳、服用维生素会有缓解，但身体出现异常的时候才会显现出来，如感冒时等等。 不要过度清洁皮肤，一定要让皮肤保持水分，再辅以锻炼加强身体的循环代谢，补充维生素，毛周角化就会减轻很多。如果是遗传性的毛周角化，是无法彻底治愈的。

日常护理

第一:   温和清洁 。  水温40度左右
        不要用含皂基或者刺激活性剂的沐浴露，同时洗澡水温也不要太高。推荐使用氨基酸沐浴露或者婴儿沐浴露。
第二： 温和去角质。 
           皮 肤觉得外界条件很差，或者受到伤害，比如干燥，风沙，去角质过度它就会产生角质保护自己，生长出“盔甲”来保留水份油脂，所以去角质要适当，力度要轻，避免搓伤皮肤。
第三： 补水保湿 。
             保护修复脆弱干燥的鸡皮肤。 
第四： 果酸尿素类产品。
            除了普通的滋润以外，还需要一些特殊护理，改善角质生长情况。毛周角化，酸不能停！但是不能用过于刺激的酸，推荐低浓度水杨酸，果酸。或者尿素。
第五: 美白，改善色沉 。
        毛周角化皮肤逐渐光滑以后，会留下色素沉淀。在皮肤比较光滑的情况下可以使用美白产品。从便宜的大创，到贵一点的only小白瓶。烟酰胺，馨肤白，white377，熊果苷，维C衍生物，甘草光定，传明酸，硫辛酸这些都是有效的美白成分。
第六:   做好防晒 ！    皮肤表面角质层被剥离，容易晒黑, 日晒会让你的毛周角化病加重！加重！加重！
